1.注册登陆
2.借款功能实现 额度1000 每天利率0.3% 每月30天计算
3.借款完成,告知用户需要还款总金额,如果分期还款,告知用户每期应还金额
4.用户可分期还款,一期固定30天,用户可以选择期数
5.用户逾期未还款,计算用户罚息,告知用户当前最小应还金额 逾期每日罚息利率0.5%
6.实现用户还款功能,(需注意:如果用户还款金额不够当前最小应还金额,则挂账)
7.实现已还金额冻结功能,借款记录进入冻结列表

借款利息= 借款金额*利率*借款天数
实际每期应还 = (借款金额+借款利息)/借款期数 
逾期罚息= 借款本金*逾期天数*逾期罚息利率, 如果两期以上逾期,则从最先逾期的那一期开始计算罚息
当期应还=(借款金额+借款利率+逾期罚息)/借款期数 -挂账总金额
挂账:账户余额的概念,入账但是不做销账处理,如果挂账金额被用掉则清0,如果没有用掉则下次还款时自动判断是否够销账使用




当期理论应还:(借款金额+借款利率+逾期罚息)/借款期数
当期实际应还:当期理论应还+上期理论应还 -挂账总金额
